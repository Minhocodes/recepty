# Recepty

Jednoduchá webová aplikace v PHP pro správu receptů.

## Funkce

- zobrazení seznamu receptů
- detail receptu
- přidání nového receptu
- smazání receptu
- filtrování podle kategorie
- živé vyhledávání v seznamu receptů
- ukládání dat do SQLite databáze

## Technologie

- PHP
- SQLite
- HTML
- CSS
- JavaScript
- Docker

## Spuštění

Projekt lze spustit přes Docker:

```bash
docker compose up --build
